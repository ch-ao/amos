Assignment 1: Create a new blank project, select a sprite from the library, then let the sprite forever move 5 steps towards mouse pointer.

Assignment 2: Create a new blank project, select a sprite from the library, set the rotation point inside the sprite and then let the sprite forever turn 30 degrees.

Assignment 3: Create a new blank project, select a sprite from the library, then let the sprite forever move 10 steps. Set the boundary for the sprite and let it move inside the display area.

Assignment 4: Create a new blank project, select 2 sprites from the library, then let one of the sprite forever move 10 steps, the other one is static. If these two sprites touch each other, stop all scripts.

Assignment 5: Create a new blank project, select a sprite with more than one style(at least two styles) from the library, code the sprite, let it forever change to next style every 0.1s.

Assignment 6: Create a new blank project, select the {diamond} from the library, code the diamond to randomly change its color every 0.2s.

Assignment 7: Create a new blank project, select a sprite from the library, then use keyboard event to control the sprite to move up, down, left and right.

Assignment 8: Create a new blank project, create a new variable, name it "life", set default value to 5 and select the "heart" icon for the variable. Then code the variable, let the variable decrease by 1 every second. Set the condition, when the variable "life" equals 0, stop all scripts.

Assignment 9: Create a new blank project, select a background image from the library, then select a sprite from library. Code the background, let the background move from top to bottom with a fixed speed.

Assignment 10: Create a new blank project, select a background image from the library, then select a sprite from library. Code the background, let the background move from right to left with a fixed speed. Then code the sprite, when [space] is pressed, let it jump with 1s (move up within 0.5s and move down within 0.5s).

Assignment 11: Create a new blank project, select a sprite from the library, then use clone event to create clones for the sprite (each 2s create a new clone). When I start as a clone, let the clone go to random position.

Assignment 12: Create a new blank project, select a sprite from the library, then use clone event to create clones for the sprite (each 1s create a new clone). When I start as a clone, let the clone go to [ random x: (-300, 300) , y: 400 ] and let it drop from top to bottom, when the clone leave the screen edge, delete the clone.

Project 1: Create a Rocket Game by yourself.

Step 2: Set the boundary for the rocket. Set rotation mode to [do not rotate].

Step 3: Set the background image. Code the background image to let it move from bottom to top(decrease y value) which seems like our rocket move forward.

Step 4: Create a barrier (select a sprite from library). Code the barrier to create the clones every 1 second. Hide the barrier. When I start as a clone, show it, set x to a radmon number between (-300, 300), set y to 300. Then let the clone fowever move with the background from top to bottom as the same speed.

Step 5: Create two variables [life] & [score] and set different icons. Set the condition. If the barrier clones touching the rocket, decrease the [life] by 1. When variable [life] = 0, stop the game. Increase the variable [score] by 1 every second.
